# 운동기록 앱 — 설치 방법

헬스장용 개인 운동기록 앱입니다. 루틴 짜고, 세트별 무게·횟수 기록하고, 휴식 타이머 돌리고, 지난 기록·달력으로 확인할 수 있어요.

**주소: https://kds335.github.io/workout-logger/**

설치라고 해도 앱스토어에서 받는 게 아니라, **웹주소를 홈 화면에 추가**하는 방식이에요. 1분이면 됩니다.

---

## 📱 아이폰 (iPhone)

> ⚠️ 꼭 **Safari(사파리)** 로 여세요. 크롬으로 열면 "홈 화면에 추가"가 안 보입니다.

1. Safari에서 주소 접속 → **https://kds335.github.io/workout-logger/**
2. 화면 아래 가운데 **공유 버튼**(□에 ↑ 화살표) 누르기
3. 메뉴 내려서 **"홈 화면에 추가"** 누르기
4. 오른쪽 위 **"추가"** 누르기
5. 홈 화면에 아이콘 생김 → 그걸로 실행하면 끝!

---

## 🤖 안드로이드 (갤럭시 등)

1. **Chrome(크롬)** 으로 주소 접속 → **https://kds335.github.io/workout-logger/**
2. 오른쪽 위 메뉴 **⋮** 누르기
3. **"앱 설치"** 또는 **"홈 화면에 추가"** 누르기
4. 확인 → 홈 화면 아이콘으로 실행하면 끝!

---

## ✅ 설치하면 좋은 점

- **와이파이·인터넷 없어도 작동** — 헬스장에서 신호 약해도 OK (한 번 열어두면 오프라인으로 돌아감)
- 일반 앱처럼 전체화면으로 깔끔하게 실행
- 무료, 회원가입·로그인 없음

## 🔒 내 기록은 안전한가요?

- 기록은 **내 폰 안에만** 저장됩니다. 서버로 안 보내고, 다른 사람도 못 봅니다.
- 다른 사람이 같은 주소로 설치해도 **각자 자기 기록**만 보여요. 절대 안 섞입니다.
- 단, 폰 하나를 둘이 같이 쓰면 그 폰 안에선 같이 보입니다(기기별로 따로예요).

## 🏋️ 쓰는 법 간단히

1. **루틴** 탭에서 "루틴 만들기" → 부위별 운동 골라서 나만의 루틴 만들기 (기본 운동 33종 미리 들어있음)
2. 루틴 **시작** → 세트마다 무게·횟수 넣고 "세트 완료" (휴식 타이머 자동, 중에 ±15초 조절 가능)
3. **기록** 탭에서 날짜별로 묶인 기록 확인 (탭하면 펼쳐짐)
4. **달력** 탭에서 운동한 날 확인 + 날짜별 루틴 계획

문제 있거나 안 되면 알려주세요!
